# 광고 파라미터 설정 가이드 (플랫폼별 실무)

> 간품 트래커 기준 — 각 광고 플랫폼 대시보드에서 어디에 무엇을 입력해야 하는지 정리

---

## 네이버 검색광고 (파워링크)

### 에어브릿지 → 간품 트래커 교체

기존에 네이버 광고에 설정된 에어브릿지 추적 URL:
```
https://abr.ge/@ganpoomclient/naver.searchad?custom_landing_url={final_url}&campaign={campaign}&sub_id={campaign_type}&ad_group={ad_group}&sub_id_2={media}&ad_creative={ad}&content={ad_extension}&term={keyword}&term_id={keyword_id}&query={query}&match={match}&sub_id_1={network}&rank={rank}&...
```

**이걸 아래 URL로 교체하면 됨:**

```
https://www.ganpoom.com/?utm_source=naver.searchad&utm_medium=cpc&utm_campaign={campaignName}&k_campaign={campaign}&k_adgroup={ad_group}&k_keyword={keyword}&k_keyword_id={keyword_id}&k_creative={ad}&k_media={media}&query={query}&network={network}&match={match}&rank={rank}
```

### 에어브릿지 파라미터 → 간품 트래커 파라미터 대응

| 에어브릿지 | 간품 트래커 | 네이버 자동치환 변수 |
|---|---|---|
| `campaign=` | `k_campaign=` | `{campaign}` |
| `ad_group=` | `k_adgroup=` | `{ad_group}` |
| `ad_creative=` | `k_creative=` | `{ad}` |
| `term=` | `k_keyword=` | `{keyword}` |
| `term_id=` | `k_keyword_id=` | `{keyword_id}` |
| `sub_id_2=` | `k_media=` | `{media}` |
| `query=` | `query=` | `{query}` |
| `match=` | `match=` | `{match}` |
| `sub_id_1=` | `network=` | `{network}` |
| `rank=` | `rank=` | `{rank}` |

> `{}` 안의 값은 네이버가 클릭 시 자동으로 실제 값으로 바꿔줌

### 설정 위치 및 단계

**네이버 광고 관리시스템** (searchad.naver.com)

1. searchad.naver.com 접속
2. 좌측 메뉴 → **캠페인 관리**
3. 광고 소재 클릭 → **수정**
4. **최종 도착 URL** 항목에서 기존 에어브릿지 URL 삭제
5. 위 간품 트래커 URL 붙여넣기 → 저장

---

## 네이버 파워컨텐츠

### 설정 위치
동일하게 searchad.naver.com → 파워컨텐츠 캠페인 → 광고 소재 → **최종 도착 URL**

```
https://www.ganpoom.com/?utm_source=naver_powercontents&utm_medium=cpc&utm_campaign={campaignName}&k_campaign={campaignName}&k_adgroup={adgroupName}&k_keyword={keyword}&k_keyword_id={keywordId}
```

## 구글 검색광고 / 디스플레이

### 설정 위치
**Google Ads** (ads.google.com)  
설정 방법이 두 가지:

---

### 방법 A — 계정 전체 적용 (최종 URL 접미사, 권장)

**ads.google.com → 캠페인 설정 → 추가 설정 → 광고 URL 옵션 → 최종 URL 접미사**

입력:
```
utm_source=google&utm_medium=cpc&utm_campaign={campaignid}&utm_content={creative}&utm_term={keyword}
```

> 계정 단위로 넣으면 모든 광고에 자동 적용됨 (가장 편함)

**또는 계정 수준 설정:**  
ads.google.com → 우상단 설정(톱니바퀴) → **계정 설정** → **자동 태그 설정** ON

자동 태그 켜면 `gclid` 파라미터가 자동으로 붙음 (우리 시스템에서 자동 수집).  
UTM과 함께 쓰려면 최종 URL 접미사도 같이 설정.

---

### 방법 B — 광고별 직접 입력

ads.google.com → 광고 → **광고 수정** → 최종 URL에 직접 입력:

```
https://www.ganpoom.com/?utm_source=google&utm_medium=cpc&utm_campaign=캠페인명&utm_term={keyword}&utm_content={creative}
```

### 구글 ValueTrack 파라미터

| 파라미터 | 의미 | 예시 결과 |
|---|---|---|
| `{keyword}` | 입찰 키워드 | 시흥간판 |
| `{matchtype}` | 매칭 유형 | e(완전) / p(구문) / b(광범위) |
| `{network}` | 네트워크 | g(구글검색) / s(검색파트너) / d(디스플레이) |
| `{device}` | 기기 | m(모바일) / t(태블릿) / c(PC) |
| `{creative}` | 소재 ID | 숫자 |
| `{campaignid}` | 캠페인 ID | 숫자 |
| `{adgroupid}` | 광고그룹 ID | 숫자 |

---

## 채널별 utm_source 값 (오타 주의)

우리 시스템에서 `utm_source` 값이 정확히 일치해야 채널 분류가 됨:

| 광고 채널 | utm_source 입력값 |
|---|---|
| 네이버 검색광고 (파워링크) | `naver.searchad` |
| 네이버 파워컨텐츠 | `naver_powercontents` |
| 네이버 서브링크 | `naver_powerlink_sublink` |
| 네이버 GFA | `naver_gfa` |
| 구글 검색 / 디스플레이 | `google` |
| 구글 앱 프로모션 | `google.adwords` |
| 텐핑 | `tenping_web` |
| 페이스북 / 인스타 | `facebook.business` |
| 카카오 검색광고 | `kakao` |

> 대소문자, 점(`.`), 언더바(`_`) 하나라도 다르면 `organic(자연유입)`으로 잘못 분류됨

---

## 주의사항

- **`ref` 파라미터는 광고 URL에 넣지 말 것**: `ref=gp001`은 CPA 에이전트 전용. 광고 URL에 넣으면 에이전트 실적으로 잘못 집계됨.
- **네이버 최종 도착 URL에 직접 입력**: UTM 파라미터는 반드시 최종 도착 URL에 붙여넣어야 유저가 실제로 ganpoom.com에 착지함. 추적 URL(별도 칸)은 3rd party 트래커 픽셀용이므로 여기엔 입력하지 말 것.
- **구글 자동태그(gclid)**: 자동태그 ON이면 UTM과 동시에 수집 가능. 끄지 말 것.
